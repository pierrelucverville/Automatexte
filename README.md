# Automatexte
Automatexte est un générateur automatique de texte flottant en français

Il fonctionne uniquement à partir de règles syntaxiques explicites
(grammaire, accords, conjugaison) et ne repose sur aucune intelligence
artificielle ni apprentissage automatique.

## Utilisation

1. Télécharger ou cloner le dépôt
2. Ouvrir le fichier `index.html` dans un navigateur
3. Cliquer sur le bouton de génération

Aucune installation, aucune dépendance, aucun serveur requis.

## Contenu

- `index.html` : interface minimale
- `moteur.js` : moteur de génération de texte
- `LICENSE` : conditions d’utilisation
